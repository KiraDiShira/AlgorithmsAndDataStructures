# Tree

A **Tree** is:

* empty, or
* a node with:
  * a key, and
  * a list of child trees.
  * (optional) a parent
  
For binary tree, node contains:
* key
* left
* right
* (optional) parent

**Height**: maximum depth of subtree node and farthest leaf
